## 项目结构

```
bovo-admin/
├── src/
│   ├── api/          # 🔌 API 接口
│   ├── components/   # 🧩 通用组件
│   ├── pages/        # 📄 页面组件
│   ├── router/       # 🛣️ 路由配置
│   ├── contexts/     # 📦 Context
│   └── config/       # ⚙️ 配置文件
├── docs/
│   ├── 开发规范/      # 📖 开发文档
│   └── 部署文档/      # 🚀 部署文档
├── k8s/              # K8s 配置
└── deploy.sh         # 部署脚本
```
