## 附录

### A. 文件命名规范

```
src/pages/cm-moxb/模块名/index.jsx     # 主页面
src/pages/cm-moxb/模块名/components/   # 子组件
src/api/modules/模块名.js              # API定义
```

### B. Git提交规范

```
feat(cm-moxb/user): 添加用户管理页面

- 实现用户列表功能
- 实现用户新增/编辑功能
- 实现用户删除功能

Closes #123
```

### C. 代码审查清单

- [ ] 代码符合项目规范
- [ ] 功能完整可用
- [ ] 无明显bug
- [ ] 有适当的注释
- [ ] 通过ESLint检查
