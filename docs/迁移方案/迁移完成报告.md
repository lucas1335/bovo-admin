## 📋 项目概述

### 项目简介

本项目是基于React 18 + Ant Design 5 + Vite构建的现代化后台管理系统，成功从Vue2技术栈迁移至React技术栈。项目采用组件化开发模式，实现了完整的业务功能模块。

### 技术栈

| 类别 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 前端框架 | React | 18.2.0 | 采用Hooks开发模式 |
| 构建工具 | Vite | 5.0.8 | 快速的开发构建工具 |
| UI框架 | Ant Design | 5.4.0 | 企业级UI组件库 |
| 路由 | React Router DOM | 6.3.0 | 客户端路由管理 |
| HTTP客户端 | Axios | 1.3.4 | API请求封装 |
| 图表 | ECharts | 6.0.0 | 数据可视化 |
| 富文本编辑器 | WangEditor | 5.1.23 | 内容编辑功能 |
| 工具库 | Day.js | 1.11.7 | 日期时间处理 |
| 加密库 | JSEncrypt | 3.5.4 | RSA加密 |

// 1. 导入API模块
import { agentList, agentDetail } from '@/api/modules/agent';

const AgentManagement = () => {
  const [data, setData] = useState([]);

  // 获取代理商列表
  const fetchAgents = async () => {
    try {
      const response = await agentList({
        page: 1,
        pageSize: 10,
        status: 1
      });
      setData(response.data);
    } catch (error) {
      message.error('获取代理商列表失败');
    }
  };

  useEffect(() => {
    fetchAgents();
  }, []);

  return (
    <Table dataSource={data} />
  );
};

## 🧭 路由配置说明

### 路由结构

项目采用嵌套路由结构，支持懒加载：

```javascript
// 主路由配置
const routes = [
  {
    path: '/login',
    component: Login
  },
  {
    path: '/',
    component: Layout,
    children: [
      // 墨小帮业务模块
      {
        path: 'cm-moxb',
        children: [
          { path: 'agent', component: Agent },
          { path: 'asset', component: Asset },
          { path: 'bank', component: Bank },
          // ... 更多子路由
        ]
      },
      // 门户管理模块
      {
        path: 'cm-portal',
        children: [
          { path: 'article', component: Article },
          { path: 'banner', component: Banner },
          // ... 更多子路由
        ]
      },
      // 系统管理模块
      {
        path: 'cm-system',
        children: [
          { path: 'user', component: User },
          { path: 'role', component: Role },
          { path: 'menu', component: Menu },
          // ... 更多子路由
        ]
      }
    ]
  }
];
```

### 路由特性

1. **懒加载**：所有页面组件采用动态导入
   ```javascript
   const Agent = lazy(() => import('@/pages/cm-moxb/agent'));
   ```

2. **权限控制**：基于用户角色的路由访问控制

3. **面包屑**：自动生成面包屑导航

4. **标签页**：支持多标签页切换
