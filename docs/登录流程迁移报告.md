## 登录流程对比

### 旧项目 (Vue) 登录流程

```
1. 用户填写登录表单
   ├── username (用户名)
   ├── password (密码)
   ├── authCode (Google验证码) ⭐
   ├── code (图片验证码)
   └── rememberMe (记住密码)

2. 提交登录请求
   ├── 接口: POST /auth/l_l
   ├── 参数: username, password, authCode, code, uuid
   └── 响应: { token: "xxx" }

3. 存储Token
   ├── Cookie: Admin-Token
   └── 加密存储用户名密码到Cookie（如果勾选记住密码）

4. 获取用户信息
   ├── 接口: GET /system/user/getInfo
   ├── 响应: { user, roles, permissions }
   └── 存储到Vuex和localStorage

5. 获取动态路由
   ├── 接口: GET /getRouters
   ├── 响应: 路由配置JSON
   └── 使用 import.meta.glob 动态加载组件

6. 跳转到首页
```

### 新项目 (React) 登录流程

```
1. 用户填写登录表单 ✅
   ├── username (用户名)
   ├── password (密码)
   ├── authCode (Google验证码) ⭐ 新增
   ├── code (图片验证码)
   └── remember (记住密码)

2. 提交登录请求 ✅
   ├── 接口: POST /auth/l_l
   ├── 参数: username, password, authCode, code, uuid
   └── 响应: { token: "xxx" }

3. 存储Token ✅
   ├── Cookie: Admin-Token (兼容旧项目)
   └── localStorage: token (备用)
   ├── 加密存储用户名密码到Cookie（如果勾选记住密码）
   └── 使用 jsencrypt 加密

4. 获取用户信息 ✅
   ├── 接口: GET /system/user/getInfo
   ├── 响应: { user, roles, permissions }
   └── 存储到AuthContext和localStorage

5. 获取动态路由 ✅
   ├── 接口: GET /getRouters
   ├── 响应: 路由配置JSON
   └── 使用 import.meta.glob 动态加载组件

6. 跳转到首页 ✅
```

### 3. Token管理 (`src/utils/auth.js`)

**兼容旧项目的Token存储**:
```javascript
const TokenKey = 'Admin-Token'; // 与旧项目保持一致

export function getToken() {
  return Cookies.get(TokenKey) || localStorage.getItem('token');
}

export function setToken(token) {
  Cookies.set(TokenKey, token);        // 存储到Cookie（旧项目兼容）
  localStorage.setItem('token', token); // 存储到localStorage（备用）
}
```

---

### 4. 认证上下文 (`src/contexts/AuthContext.jsx`)

**更新登录逻辑**:
```javascript
const login = async (loginData) => {
  // 1. 调用登录接口
  const response = await apiService.post("/auth/l_l", {
    username: loginData.username,
    password: loginData.password,
    authCode: loginData.authCode,  // ⭐ Google验证码
    code: loginData.code,
    uuid: loginData.uuid,
  });

  // 2. 获取token（兼容旧项目格式）
  const token = response.token || response.data?.token;
  setToken(token);

  // 3. 获取用户信息
  const userInfoResponse = await apiService.get("/system/user/getInfo");
  const { user, roles, permissions } = userInfoResponse;

  // 4. 存储认证信息
  setUser(user);
  setRoles(roles);
  setPermissions(permissions);

  return { success: true, user };
};
```
