# 借贷产品管理模块

## 概述
借贷产品管理模块提供了完整的借贷产品CRUD功能，包括产品列表展示、新增、编辑、删除、搜索等功能。

## 文件结构
```
src/pages/cm-moxb/loan/
├── index.jsx          # 模块入口文件
├── product.jsx        # 借贷产品管理页面
└── README.md          # 模块说明文档
```

## 功能特性

### 1. 产品列表展示
- 显示所有借贷产品信息
- 支持按金额范围搜索
- 支持按状态筛选
- 支持排序功能

### 2. 产品信息管理
- **最小贷款额度**：产品支持的最小贷款金额（USDT）
- **最大贷款额度**：产品支持的最大贷款金额（USDT）
- **贷款周期**：贷款周期描述（如：30天）
- **日利率**：贷款日利率百分比
- **还款类型**：还款方式（到期一次还本息）
- **还款机构**：还款机构信息
- **状态**：产品是否开启（已开启/未开启）
- **是否冻结**：产品是否被冻结
- **备注**：产品备注信息

### 3. 产品状态管理
- **状态切换**：
  - 未开启（status=0）
  - 已开启（status=1）
- **冻结状态**：
  - 否（isFreeze=1）
  - 是（isFreeze=2）

### 4. 利率配置
- 支持配置日利率
- 精度：4位小数
- 最小值：0

### 5. 期限配置
- 支持自定义贷款周期
- 文本形式描述（如：30天、60天等）

## 使用方法

### 1. 路由配置
在后台管理系统的菜单管理中添加以下路由：
- 路径：`/cm-moxb/loan/product`
- 组件路径：`cm-moxb/loan/product`
- 菜单名称：借贷产品管理

### 2. 权限配置
需要配置以下权限标识：
- `bussiness:load:product:list` - 查看列表
- `bussiness:load:product:add` - 新增产品
- `bussiness:load:product:edit` - 编辑产品
- `bussiness:load:product:remove` - 删除产品

### 3. API接口
所有接口定义在 `src/api/modules/loan.js`：
- `getLoanProductList` - 获取产品列表
- `saveLoanProduct` - 新增产品
- `updateLoanProduct` - 更新产品
- `deleteLoanProduct` - 删除产品

## 技术实现

### 使用的组件
- **CmBasePage**：基础列表页面组件，提供表格、搜索、分页功能
- **DataForm**：表单组件，提供新增/编辑表单功能
- **Ant Design**：UI组件库

### 表单验证
- 最小贷款额度：必填
- 最大贷款额度：必填
- 贷款周期：必填
- 日利率：必填
- 还款类型：必填
- 还款机构：必填

### 数据格式
- 金额保留2位小数
- 利率保留4位小数
- 状态使用字符串格式（'0'/'1'）
- 冻结状态使用字符串格式（'1'/'2'）

## 业务规则

1. **金额范围验证**：最大贷款额度应大于最小贷款额度
2. **利率范围**：日利率不能为负数
3. **状态切换**：只有已开启的产品才能被用户使用
4. **冻结规则**：被冻结的产品不可用

## 扩展功能

如果需要扩展以下功能，可以基于现有代码进行修改：

3. **期限选项**：改为Select选择固定期限
